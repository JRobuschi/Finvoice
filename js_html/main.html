    ¿Qué significa HTML, y cuál es su estructura básica?

    HTML significa "HyperText Markup Language", este es un lenguaje de marcado que se utiliza para crear páginas web, La esctructura basica que requiere un documento de HTML es de una serie de etiquetas que se abren y se cierran, las mismas se utilizan para definir la estructura y el contenido de la pagina. La etiqueta HTML principal es !DOCTYPE html, que indica que el documento está escrito en HTML5. Dentro del !DOCTYPE html, se encuentran las etiquetas html, head y body. La etiqueta html contiene todo el contenido del documento HTML, la etiqueta head se utiliza para incluir metadatos como el título de la página, y la etiqueta body se utiliza para incluir el contenido visible de la página.
